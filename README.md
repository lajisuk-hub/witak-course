# 국공립 신규위탁 과정 진행

신규위탁에 참여하는 원장님들이 **우리 지자체 공고문에 맞춘 문서 뼈대**를 자동으로 받는 도구.

## 하는 일

1. 수강생이 우리 지자체 **공고문(PDF/한글 hwpx)** 을 올린다
2. AI가 **제출서류 목차**를 뽑고, 진행자가 등록해 둔 **샘플 꼭지**와 짝을 지어준다
3. 화면에서 목차를 확인·수정한다
4. **내 지자체 전용 문서 뼈대**가 만들어진다 (샘플 있음 / 직접 작성 표시)
5. **한글(.hwpx)** 파일이나 PDF로 내려받는다

## 화면

- `/` 수강생 화면
- `/admin` 진행자(원장님) 화면 — 문서 샘플 꼭지 등록 (비밀번호 `1234`)

## 저장

수강생이 쓴 내용은 **본인 컴퓨터(브라우저)** 에만 저장된다. 서버 비용 0원.

## 환경변수

| 이름 | 설명 |
|---|---|
| `ANTHROPIC_API_KEY` | 공고문에서 목차를 뽑을 때 사용 |

## 한글(.hwpx) 만들기 방식

`public/base.hwpx` 는 글자모양·문단모양 정의(`Contents/header.xml`)만 남긴 빈 한글 문서다.
`lib/hwpx.js` 가 본문(`Contents/section0.xml`)만 새로 만들어 끼워 넣는다.

주의 (과거 교훈):
- `mimetype` 은 **압축하지 않고 맨 앞**에 넣어야 한글이 연다
- `linesegarray` 는 넣지 않는다 (한글이 다시 계산한다)

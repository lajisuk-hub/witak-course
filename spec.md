# 국공립 신규위탁 과정 진행 프로그램 — 만들 것 정리표

- 폴더/레포 이름: `witak-course`
- 정리한 날: 2026-07-21

## 1. 한 줄 설명
국공립 신규위탁에 참여하는 원장님들이 **자기 지자체 목차에 맞춰 자동으로 짜인 문서 뼈대**를 받고,
차시를 순서대로 따라가며 칸을 채우면 **한글(HWPX) 통합서류 한 부 + 발표 PPT**가 완성되는 도구.

지금 카톡/전화로 일일이 안내하고, 수강생마다 목차를 다시 정리하던 번거로움을 없앤다.

## 2. 쓰는 사람
- **수강생**(신규위탁 참여 원장님들, 여러 분) — 링크를 받아 각자 진행
- **원장님(진행자)** — 관리자 화면에서 문서 샘플 한 번 등록

## 3. 핵심 흐름 (이게 이 프로그램의 심장)

```
[원장님] 문서 샘플을 꼭지별로 등록  ← 한 번만. 모든 기수에 재사용
              ↓
[수강생] 우리 지자체 공고문 업로드  →  AI가 목차 추출  →  화면에서 확인·수정
              ↓
[사이트] 원장님 샘플 꼭지를 그 목차 순서대로 자동 배치
         · 목차에 있고 샘플에도 있음 → 샘플 내용을 넣어둠
         · 목차에 있는데 샘플에 없음 → "직접 작성" 빈칸으로 표시
         · 샘플에 있는데 목차에 없음 → "이번엔 안 씀"으로 빼둠
              ↓
        ★ 내 지자체 전용 문서 뼈대(목차) 완성 ★
              ↓
[수강생] 차시 순서대로 칸 채우기 (아래 4번)
              ↓
[사이트] 채운 내용이 그 뼈대에 계속 쌓임  →  한글(.hwpx) 통합서류 + PDF
              ↓
[면접 코칭] 완성 PDF를 원장님께 전달 → 코칭
              ↓
[발표자료] 문서 내용을 바탕으로 발표 PPT 만들기
```

## 4. 차시 구성 (제공하신 일정표 그대로)
| 차시 | 내용 | 주의할 점 | 활용할 기존 프로그램 |
|---|---|---|---|
| 0 | **지자체 목차 만들기** (신규) | 공고문 업로드 → 목차 확정 | — |
| 1 | 자기소개서 작성하기 | A4 한 장, 원장 자격이 충분하다는 내용 | (신규 제작) |
| 2 | 40인 기준 회계서류 (세입·세출) | 반별 구성 확정 후 작성, 회계 1:1 코칭 | **예산서 자동화** |
| 3 | 연간–월간–하루일지 계획 | 개정 표준보육과정, 놀이중심, 해당연령 | **프로그램 만들기** |
| 4 | 특색프로그램 계획 | 영아/유아 구분, 트렌드+시군구 요소 | **프로그램 만들기** |
| 5 | 취약보육 계획 | 3가지 구성, 시군구 추천 유형 | **취약보육** |
| 6 | 학부모 참여수업 계획 | 아빠·조부모 등 3가지 이상 | (신규 제작) |
| 7 | 전체문서 나열하고 내용 보충 | 샘플 제공, 사진은 직접 준비 | — |
| 8 | 목차 및 표지 만들고 수정보완 | 문서 완성 후 목차 구성 | — |
| 9 | **면접 코칭용 PDF 만들기** | 완성 문서를 PDF로 | — |
| 10 | **발표자료(PPT) 만들기** | 심사 발표용 | **재위탁앱 11번 발표자료** |

기간은 기수마다 바뀌므로 **시작일만 넣으면 자동으로 다시 계산**된다.

## 5. 차시 화면 (모든 차시 공통 4칸)
1. 이번 차시 할 일 + 주의할 점 (원장님이 쓰신 안내 그대로)
2. 강의 녹화본 / 패들릿 자료 바로가기
3. 인터뷰식 입력칸 (질문에 답하듯 채우기)
4. AI 다듬기 → 결과 확인·수정 → 저장(완료 표시) → **내 문서 뼈대에 자동 반영**

## 6. 정해진 사항
| 항목 | 정한 것 |
|---|---|
| 샘플 등록 | **관리자 화면에서 꼭지별로 붙여넣기** (비밀번호 보호) |
| 목차 올리기 | **공고문 파일 업로드 → AI가 목차 추출 → 확인·수정 화면** |
| 저장 방식 | 수강생 본인 컴퓨터에만 (브라우저 저장) — 비용 0원 |
| 진도 확인 | 원장님이 남의 진도 보는 기능 **없음** (2차로) |
| AI 사용 | `claude-sonnet-5` |
| 결과물 | 한글(.hwpx) 통합서류 + PDF + 발표 PPT |
| 배포 | Vercel 링크 |
| 참고할 기존 앱 | gongmun-maker(HWPX), wmentor-recommission(단계·발표자료), daycare-branding(단계 인터뷰) |

## 7. 한글(HWPX) 통합서류
- 제가 **자리표시자가 들어간 가벼운 hwpx 틀**을 새로 만든다
- 목차가 사람마다 다르므로, **꼭지 개수·순서에 맞춰 문단을 만들어 넣는 방식**으로 만든다
- 수강생은 받은 파일을 한글로 열어 **표지·사진·표 서식만 다듬으면 끝**
- 기술 근거: 공문 만들기 앱에서 검증된 방식 (mimetype 무압축 유지 + `linesegarray` 제거)

## 8. 만드는 순서 (제안)
- **1차** — 0차시(목차 자동 생성) + 관리자 샘플 등록 + 문서 뼈대 화면 + HWPX 내려받기
  → 이것만으로도 "매번 다시 정리하는" 번거로움이 사라짐
- **2차** — 1·6·7·8차시 (신규 입력 화면) + PDF
- **3차** — 2·3·4·5차시 (기존 프로그램 이식: 예산서·프로그램·취약보육)
- **4차** — 10차시 발표자료(PPT)

## 9. 나중에 (미룸)
- 수강생 진도를 원장님이 모아 보는 관리 화면
- 어느 컴퓨터에서든 이어서 하기 (로그인)
- 기존 위탁운영계획서 디자인 서식 그대로 살리기

## 10. 아직 안 정한 것 (추천안으로 진행)
- 차시별 입력 질문 문항 → 추천: 원장님의 '주의할 점'과 샘플 서류를 근거로 제가 초안 작성 *(고쳐주세요)*
- 패들릿·녹화본 링크 → 추천: 차시마다 버튼 하나. 주소는 알려주시면 교체
- 첨삭 요청 → 추천: 1차는 "완성본을 원장님께 카톡으로 보내기" 안내 문구만
